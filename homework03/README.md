In this repository, each exercise corresponds to the exercise in the homework 3 listed on the class website.
Exercise 1 reads in a JSON file into python and has three functions which each do a unique thing (total mass, large mass, and non eukaryotes).
Exercise 2 reads in a JSON file into python and then converts that into a CSV file according to the homework directions.
Exercise 3 reads in a JSON file into python and then converts that into a XML file according to the homework directions.
Exercise 4 reads in a JSON file into python and then converts that into a YAML file according to the homework directions.
